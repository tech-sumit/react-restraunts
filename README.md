# react-restraunts
## Installation

Simply run following commands

## Commands
## For repo clone:
    $ git clone https://github.com/tech-sumit/react-restraunts
    $ cd react-restraunts
    $ npm install
    $ npm start

## Docker:
    $ docker pull mrsumit/restaurants
    $ docker run -p 8000:8000 mrsumit/restaurants
